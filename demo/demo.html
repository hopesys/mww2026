Tech Stack & Architecture

Frontend

  • Next.js 14 (App Router): โครงสร้างมาตรฐาน

Infrastructure

AI Development Roadmap (Vibecode)

ขั้นตอนการพัฒนาแบบ Step-by-Step พร้อม Prompt ที่เตรียมไว้ให้แล้ว สามารถกด Copy เพื่อนำไปสั่งงาน AI ได้ทันที

Phase 1: Landing Page & Branding

สร้างหน้าแรกที่สวยงาม ดึงดูดสายตา เน้นธีม Wellness

v0.dev Claude Sonnet
"Create a modern, elegant landing page for 'Miss Wellness'. Use a color palette of soft greens, cream, and gold to represent health and prestige. Include sections for: Hero banner with video background, About the pageant, Timeline, Sponsors, and a sticky 'Apply Now' / 'Vote' button. Use Tailwind CSS and Lucide React icons."

Phase 2: Registration System

ระบบรับสมัครแบบ Multi-step พร้อมอัปโหลดรูปภาพลง Cloud

"Create a multi-step registration form for candidates using React Hook Form and Zod for validation.
Step 1: Personal Info (Name, Age, Contact).
Step 2: Wellness Concept (Essay box).
Step 3: Photo Upload (Upload file to Supabase Storage bucket 'candidates' and get public URL).
Submit: Create a new record in 'candidates' table. Ensure specific error handling."

Phase 3: Voting System & Payment

ระบบโหวตแบบ Transactional และการตัด Point อย่างปลอดภัย

"Implement a voting mechanism.
1. Create a server action `voteForCandidate(candidateId, points)`.
2. Check if current user has enough `points_balance`.
3. Use a Supabase Transaction (RPC) to: deduct points from user, increment `vote_count` for candidate, and log into `votes` table atomically.
4. If insufficient balance, redirect to payment page."
